**零食商城购物微信小程序**
***
### 功能一览
+ 首页
+ 分类
+ 购物车（缓存）
+ 个人中心
+ 商品详情
+ 加购付款（实现单选全选多选）
+ 订单管理（分页）
+ 地址管理
